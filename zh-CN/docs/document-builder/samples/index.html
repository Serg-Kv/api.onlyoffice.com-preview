构建器框架示例

使用 C++、.Net、Python 和 Java 的指南来运行示例。

填充电子表格​

使用基于数组的值填充电子表格。

参见：C++ / .Net / Python / Java

注释电子表格错误​

为带有公式错误的电子表格单元格添加注释。

参见：C++ / .Net

创建演示文稿​

创建包含多张幻灯片、图片和文本的演示文稿。

参见：C++ / .Net / Python / Java

创建图表演示文稿​

创建单张幻灯片的图表演示文稿。

参见：C++ / .Net

创建基本表单​

创建包含文本和图像输入的基本表单。

参见：C++ / .Net / Python / Java

创建高级表单​

创建具有表格结构的高级表单。

参见：C++ / .Net

填充表单​

用文本值和图像填充表单。

参见：C++ / .Net

生成用户反馈报告​

创建包含平均分数、突出显示的负面评论、个人评分和图表的 XLSX 报告。

参见：C++ / .Net / Python / Java

从订单数据生成发票​

创建包含客户详情、逐项订单、定价和含税总额的 PDF 发票。

参见：C++ / .Net / Python / Java

生成雇佣协议​

根据人力资源表单数据创建个性化的 PDF 雇佣协议，并将其保存到 DocSpace 以供签署和存储。

参见：C++ / .Net / Python / Java

生成个性化投资计划​

创建根据客户财务目标和输入数据定制的包含预测和图表的 XLSX 投资计划。

参见：C++ / .Net / Python / Java

生成库存报告​

创建包含当前库存项目（包括数量和状态指示器）的 XLSX 报告。

参见：C++ / .Net / Python / Java

创建初创企业市场分析演示文稿​

使用集成的外部数据生成包含市场趋势、竞争情况和客户洞察的演示文稿。

参见：C++ / .Net / Python / Java

创建商业报价​

生成 DOCX 格式的个性化商业报价，包含客户特定的产品、服务和时间表。

参见：C++ / .Net / Python / Java

创建员工培训计划​

生成结构化的 DOCX 培训计划，包含每位员工的学习目标、截止日期和资源。

参见：C++ / .Net / Python / Java

创建年度业务报告​

根据外部数据生成包含财务业绩、主要成就和战略亮点的 DOCX 年度报告。

参见：C++ / .Net / Python / Java
