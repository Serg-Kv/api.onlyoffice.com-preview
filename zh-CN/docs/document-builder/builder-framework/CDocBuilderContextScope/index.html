CDocBuilderContextScope

ONLYOFFICE Document Builder 使用的栈分配类，用于为本地作用域内执行的所有操作设置执行上下文。当调用 builder 的 CloseFile 方法时，所有打开的作用域将自动关闭。

语法​

class CDocBuilderContextScope

方法​

名称描述
Close关闭当前作用域。
备注

Java 使用驼峰命名法的方法名：close。
