变更日志

ONLYOFFICE 文档 API 的更改列表。

版本 9.4​

  • 新增插件命令日志功能，用于在浏览器控制台中启用插件命令的调试输出。
  • 添加了 editorConfig.plugins.disable 参数，用于在加载时阻止指定插件。
  • 在支持的界面语言列表中添加了克罗地亚语 (hr)。
  • 移除了已弃用的 editorConfig.customization.commentAuthorOnly 字段。
  • 为 onStartFilling 事件添加了 roles 参数，包含角色和用户信息。
  • 修复了 destroyEditor 方法中的内存泄漏问题，该问题导致无法完全释放内存。

版本 9.1​

  • 如果文档无法在 WOPI 中被锁定，则文档将以查看器模式打开并显示错误消息。
  • 在 CheckFileInfo WOPI 操作中添加了 UserCanOnlyComment 属性。
  • editorConfig.customization.uitheme 参数现已可用于移动编辑器。
  • 新增对 hml 格式的打开支持。
  • 新增从 pptx 格式到 txt 的转换。
  • 将 editorConfig.customization.logo.image 的尺寸要求更改为 300x20。

版本 9.0​

  • 添加了 editorConfig.customization.suggestFeature 参数。
  • 更改了 editorConfig.customization.macros：在 9.0.3 版本中，此参数将完全禁用运行、添加和编辑宏（而不仅仅是自动启动）。
  • editorConfig.customization.toolbarHideFileName 参数现已可用于移动编辑器。
  • 在 editorConfig.customization.uiTheme 参数中新增浅色主题和深色主题的主题 ID。
  • 新增对 odg 格式的打开支持。
  • 新增对 md 格式的打开支持。
  • 新增预加载编辑器静态资源的能力。
  • 针对简体中文界面，新增 editorConfig.customization.forceWesternFontSize 参数。
  • 新增 editorConfig.customization.layout.header.user 参数。
  • 新增对 vsdm、vsdx、vssm、vssx、vstm、vstx 格式的转换能力。
  • 在 文档类型 参数中新增 diagram 文档类型。
